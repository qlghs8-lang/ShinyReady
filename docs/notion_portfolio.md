## 데모 영상

> Notion 무료 요금제는 파일 업로드당 5MB 제한이 있어, 원본 GIF(최대 47.9MB)를 MP4로 재인코딩했습니다.
> 변환 파일은 `ShinyReady_PlayGIF\mp4_out\` 폴더에 있으며 모두 5MB 이하입니다 (1.4MB~3.3MB).
> 아래 각 항목 아래에 `mp4_out` 폴더의 동명 `.mp4` 파일을 드래그&드롭으로 첨부하세요.

### 1. 기본 세차 루프 — 세제 보충 & 코인 획득

*(여기에 `보급소에서 세제를 충전 후 키오스크에 충전, 들어온 차에 세차를 진행 후 코인 획득.mp4` 첨부 · 2.59MB)*

## Notion에 올리는 방법

1. Notion에서 새 페이지 생성
2. 페이지 메뉴(`···`) → **Import** → 이 파일(`notion_portfolio.md`) 선택
   - 또는 새 페이지에서 이 문서 내용을 복사해 붙여넣기
3. 각 "여기에 ~ 첨부" 위치에 `바탕화면\ShinyReady_PlayGIF\mp4_out\` 폴더의 동명 `.mp4` 파일을 드래그&드롭
4. 필요 시 캡션 문구를 자유롭게 수정
